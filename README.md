# 🌿 Aarunya Blog Website

Aarunya is a responsive blog website built using HTML, CSS, and JavaScript.

## ✨ Features

- 📝 Create Blog
- ✏️ Edit Blog
- 🗑️ Delete Blog
- 🔍 Search Blogs
- 🌙 Dark Mode
- 📄 Export Blogs as PDF
- 💾 Local Storage
- 📱 Responsive Design
- 🔢 Character Counter
- ⏱️ Reading Time

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- jsPDF

## 🚀 How to Use

1. Open the website.
2. Enter a blog title and content.
3. Click **Publish Blog** to create a new blog.
4. Use the **Search** box to find blogs.
5. Click **Edit** to update a blog.
6. Click **Delete** to remove a blog.
7. Switch between **Light** and **Dark Mode** using the theme button.
8. Click **Export PDF** to download all blogs as a PDF.
